A query builder for SQLite3 based on Knexjs

## Installation

### Via [PyPi](https://pypi.org/project/knexpy)

```bash
pip install knexpy
```

### Via Source

```bash
git clone https://github.com/carlossilva2/Knexpy.git
cd Knexpy
pip install .
```
